# Somisha Adak — Portfolio

Static portfolio website showcasing Somisha Adak — built with plain HTML, CSS, and JavaScript.

## What’s included

- `index.html` — main page and content
- `styles.css` — styling, themes, and animations
- `script.js` — interaction logic (theme toggle, reveal animations, sounds)
- `profile.jpg` — profile photo (replace as needed)
- `resume.pdf` — optional resume file (place in project root to enable embedded preview)

## Local preview

Open `index.html` directly, or run a simple HTTP server for full functionality (recommended for iframe resume preview):

```bash
# From the project root
python -m http.server 8000
# Then visit http://localhost:8000
```

## Notes

- Add `resume.pdf` to the project root to enable the inline resume preview.
- Theme preference is stored in `localStorage` and persists across reloads.
